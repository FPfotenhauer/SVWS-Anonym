## Name Lists

This repository includes German name lists for anonymization purposes in the `data/` directory:

- **nachnamen.json**: 2000 German last names
- **vornamen_m.json**: 500 German male first names  
- **vornamen_w.json**: 500 German female first names

These name lists are sourced from the [JSON-Namen repository](https://github.com/FPfotenhauer/JSON-Namen) and are used to replace real names in database entries while maintaining data consistency.
